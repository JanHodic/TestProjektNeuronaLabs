# Neurona Lab – Patient Management Demo

Full-stack aplikace pro správu zdravotnických dat pacientů.

## Stack

- Backend: .NET 9, HotChocolate GraphQL, EF Core, PostgreSQL
- Frontend: Next.js + React + TypeScript, Tailwind CSS, Material UI
- Docker: docker-compose
